# Muhammad Ahmed — Cinematic Portfolio

Production-ready personal portfolio with cinematic scroll, Three.js atmospheres, GSAP motion, and a modern Next.js stack.

## Stack

- Next.js 16 (App Router) + React 19 + TypeScript
- GSAP + ScrollTrigger
- Three.js
- Framer Motion
- Tailwind CSS v4 + CSS Modules

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Content

Personal data lives in `data/profile.ts` and `data/content.ts`.
Profile image: `public/images/profile.jpg`.

## Deploy

Optimized for Vercel.
